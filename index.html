🌐 วิธีนำแอปนี้ขึ้น GitHub Pages เพื่อเปิดใช้งานออนไลน์

คุณสามารถอัปโหลดหน้าเว็บนี้ขึ้น GitHub เพื่อเปิดใช้งานออนไลน์เป็น Link ส่วนตัวของคุณเองได้ฟรี มีขั้นตอนดังนี้:

  1. สมัครสมาชิก หรือ Login ใน GitHub.com
  2. สร้าง Repository ใหม่ (กดปุ่ม New) ตั้งชื่อตามต้องการ เช่น magazine-extractor และตั้งสถานะเป็น Public
  3. อัปโหลดไฟล์ index.html นี้ขึ้นไปใน Repository ที่เพิ่งสร้างขึ้นมา
  4. ไปที่เมนู Settings ของ Repository ของคุณ
  5. เลือกเมนู Pages ที่แท็บด้านซ้าย
  6. ในหัวข้อ Build and deployment ตรงส่วน Branch ให้เปลี่ยนจาก None เป็น main (หรือ master) แล้วกดปุ่ม Save
  7. รอประมาณ 1-2 นาที คุณจะได้ลิงก์หน้าเว็บออนไลน์สำหรับเปิดใช้ผ่านมือถือ หรือคอมพิวเตอร์ที่ไหนก็ได้ทันที!
🔒 ความปลอดภัย: ไม่ต้องกังวลเรื่องการโดนดักจับ API Key เนื่องจากแอปนี้ทำงานแบบ Serverless ข้อมูล Key และประวัติข่าวสารจะถูกเก็บไว้เฉพาะใน Browser (Local Storage) ของคุณเองเท่านั้น ไม่มีการส่งไปที่เซิร์ฟเวอร์ใด ๆ ทั้งสิ้นยกเว้นระบบวิเคราะห์ของ Google Gemini โดยตรง
